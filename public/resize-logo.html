This will resize your logo.png to 192x192 and 512x512 icon sizes.

Instructions:

  1. Select your logo.png file (or click the button to use logo.png from the same folder)
  2. Click the resize button
  3. Two PNG files will download automatically
  4. Save them in the public/public/ folder as icon-192.png and icon-512.png
  5. Rebuild: npm run build
